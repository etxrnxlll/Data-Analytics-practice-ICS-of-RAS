# 🛠 Задание на практику: Аналитический дашборд в Power BI

> 📌 **Цель:** Создать интерактивный отчёт для анализа финансового состояния организаций на основе многолетних данных с использованием Microsoft Power BI, DAX и Power Query.

---

## 📥 1. Источники данных
Подключите следующие файлы (формат `Excel` / `CSV`):
- 📊 `ORG` — справочник организаций (ИНН, наименование, ОКВЭД, регион)
- 📅 Годовая отчётность за `2021–2025` гг.
- 👥 Численность персонала
- 💰 Доходы / расходы
- 🏛 Налоги
- 📍 Географические координаты (`lat`, `lon`)

> ⚠️ **Важно:** Поле `ИНН` должно храниться как **текст**, чтобы сохранить ведущие нули и избежать ошибок при связывании таблиц.

---

## 🔄 2. Трансформация данных (Power Query)
| Действие | Описание |
|:---------|:---------|
| 📏 Приведение размерности | Перевод доходов/расходов из `руб.` → `тыс. руб.` (деление на `1000`) |
| 🔤 Обработка ИНН | Установить тип данных `Текст`, удалить лишние пробелы и скрытые символы |
| 🗑 Фильтрация выбросов | Исключить организации из списка «неадекватных» (по совпадению `ИНН`) |
| 🔗 Объединение таблиц | Связать все источники по ключам `ИНН` и `Год` (рекомендуется модель «звезда») |

---

## 📐 3. Расчётные метрики (DAX)
Все показатели реализуются как **меры** (Measures), а не вычисляемые столбцы:

| Метрика | Формула (DAX) |
|:--------|:--------------|
| 💹 Рентабельность | ``DIVIDE([Чистая прибыль], [Выручка], 0)`` |
| 💧 Текущая ликвидность | ``DIVIDE([Оборотные активы], [Краткосрочные обязательства], 0)`` |
| 👷 Производительность труда | ``DIVIDE([Выручка], [Численность], 0)`` |
| 📈 Темп роста (YoY) | ``CALCULATE([Выручка], SAMEPERIODLASTYEAR('Календарь'[Дата]))`` |

> 💡 **Рекомендация:** Используйте `DIVIDE()` вместо оператора `/` для безопасного деления и защиты от ошибок `#DIV/0!`.

---

## 📊 4. Визуализации
| Тип визуализации | Назначение |
|:-----------------|:-----------|
| 📋 Матрица / Сводная таблица | Группировка по `ОКВЭД` и `Регион`, отображение сумм и средних значений |
| 📉 Линейный график | Динамика показателей по годам (с разбивкой по отраслям) |
| 🗺 Карта (Bing Maps / Shape Map) | Отображение организаций по координатам (`lat`, `lon`), цветовое кодирование по эффективности |
| 🎛 Срезы (Slicers) | Интерактивные фильтры по `Год`, `Регион`, `Отрасль`, `Тип организации` |

---

## 🖥 5. Финальный дашборд
- [ ] Объединить все визуализации на **одной или нескольких логически связанных страницах**
- [ ] Настроить **взаимодействие** между элементами (клик по карте фильтрует таблицы и графики)
- [ ] Добавить **KPI-карточки**: общая выручка, средняя рентабельность, количество организаций
- [ ] Обеспечить **интуитивную навигацию**: кнопки перехода между страницами, закладки, всплывающие подсказки
- [ ] Проверить корректность расчётов при различных комбинациях фильтров

> ✅ **Критерий готовности:** Отчёт должен быть полностью интерактивным, методологически корректным и готовым к демонстрации руководителю.
